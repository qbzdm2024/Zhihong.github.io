Step 1 — Auto-Retrieve Full Texts

Tries Unpaywall → Semantic Scholar → OpenAlex → Europe PMC for each record. Gets PDFs where available, otherwise extracts HTML full text (saved as .txt). A PRISMA snapshot is saved before the run begins.

Step 1b — Retry Failed Records (arXiv · CORE · Direct URL)

Re-attempts download for all — records that could not be retrieved automatically. Uses additional sources: arXiv (very effective for CS/AI papers), CORE open-access aggregator, and the paper's direct URL.
